User Experience Design

Essentials of User-Centered Design

Empathize

The first step is to empathize with in the target group. It is of great importance since otherwise my own assumptions can be subjective and could compromise the design process. In this process it is necessary to research the current target audience and to immerse myself with the end user.

Define

This step is important for determining the "problem". What needs to be done is defined in this phase. This is based on the user needs from empathizing with end user.

Ideate

From the information gathered from the previous steps, we can start generating ideas. To brainstorm and to start devising creative solutions to solve the problem. By the end of this step, a concept should have been devised.

Prototype

Based from the concept, a prototype can start to be developed. A prototype does not necessarily have to represent the end product and is used to determine whether if the solution(s) will work well or not for the problem.

Test

After developing the prototype and determining the specific target audience. A test can be commenced to determine the efficacy of the prototype. If the outcome result does not satisfy, we can go back to the previous phases and to iterate until eventually the result gets closer to the desired outcome.
