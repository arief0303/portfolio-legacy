Interactive ThreeJS Visual Art

Sprint 2

As part of the process for me to practice my Javascript skills i decided to make an interactive visual that is now playing on the background of this site by using ThreeJS web 3D graphics library.

Link to the interactive visual.

In this article i will describe in the detail of the steps i took to create this ThreeJS interactive visual.

1. Creating the scene

The way i created this interactive visual is first to define the basic components to create the whole scene such as:

  • the scene itself
  • a camera to see the scene
  • a renderer to display the scene unto the screen
// scene
var scene = new THREE.Scene();

// camera
var camera = new THREE.PerspectiveCamera(80, window.innerWidth / window.innerHeight, 0.1, 1000);

// renderer
var renderer = new THREE.WebGLRenderer({
antialias: true,
alpha: true
});
renderer.setSize(window.innerWidth, window.innerHeight); //set size of the renderer to window size
document.body.appendChild(renderer.domElement); //create canvas on the html body

// prism object
  var geometry1 = new THREE.TetrahedronGeometry(1.5, 0); //object1 geometry
  var material1 = new THREE.MeshNormalMaterial(); //object1 material

2. Interactive Controls

Then i use an additional ThreeJS library called OrbitControls.js to add interactive camera controls for the user to navigate around the 3D space. Then i also modify a few parameters of the OrbitControls to make it pan smoothly as the user navigates around.

// controls
var controls = new THREE.OrbitControls(camera, renderer.domElement);
controls.autoRotate = true;
controls.autoRotateSpeed = 1;
controls.enablePan = true;
controls.enableZoom = true;
controls.enableDamping = true;
controls.dampingFactor = 0.07;
controls.rotateSpeed = 0.8;

By this point, we would be having a basic scene that the user can navigate around the object.


3. Inheritance Cloner

I wanted to have a very dynamic visuals to fills the whole screen and i spent a lot of time experimenting on this. From my experience from using 3D software suite such as Cinema 4D and Blender, i know that 3D object can be 'cloned' using a cloner tool. So with the help of Mr.Chris Geene i was mentored of how to create an 'inheritance cloner' by creating multiple meshes of the object using a for-loop then adding each meshes to an object array then to use an if statement to link every newly created meshes to its predecessor. 


// cloner
var objArray = [];
for (let i = 0; i <= 16; i++) {
var meshes = new THREE.Mesh(geometry1, material1);
meshes.position.set(i - 5, i, i);

if (i == 0) {
scene.add(meshes);
var primeMesh = meshes;
objArray.push(meshes);
} else {
primeMesh.add(meshes);
primeMesh = meshes;
objArray.push(meshes);
}
    }

4. Animating the objects to move based of the 'Golden Ratio'

The last step is to animate the objects, the goal i had in mind is to make the objects move(in rotation) with some certain flow. So i used Math.PI to make the objects rotate according to the golden ratio.


function animate() {
    requestAnimationFrame(animate);
  
    controls.update();
    
    // golden ratio animation
    var div = 512;
    for (var x = 0; x < objArray.length; x++) {
      objArray[x].rotation.x += Math.PI / 2;
      objArray[x].rotation.y += Math.PI / 4;
      objArray[x].rotation.z += Math.PI / 4;
    }
  
    renderer.render(scene, camera);
};
